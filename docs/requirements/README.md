# Requirements

Thư mục này dùng để lưu user story, use case, rule nghiệp vụ và các tài liệu yêu cầu đầu vào của dự án.

- `marketplace_rules.md`: Bộ rule nghiệp vụ đã chốt cho project, bid, contract, milestone, message, review, payments & Flyway.
- `project_completion_plan.md`: Kế hoạch hoàn thiện dự án theo phase, ưu tiên nghiệp vụ, quality gate và tiêu chí bàn giao.
- `demo_runbook.md` & `official_endpoint_contract.md`: Đã cập nhật với search, transaction, realtime và migration.
